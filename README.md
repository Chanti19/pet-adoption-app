
GetPet – Pet Adoption Web App

A simple and attractive pet-adoption website where users can browse pets, filter them, and adopt them using real APIs.

-------------------------------------------------------------
HOW TO RUN THE PROJECT

1) Download the project
   - Download ZIP from GitHub
   - Extract the folder

2) Install Node.js
   - Go to https://nodejs.org
   - Download LTS version
   - Install (Next → Next → Finish)

3) Open project folder in terminal
   - Right-click → Open in Terminal

4) Install required packages
   npm install

5) Start the server
   node server.js

6) Open browser and visit:
   http://localhost:3000

-------------------------------------------------------------
PROJECT STRUCTURE
- index.html  → Main homepage
- login.html  → Login page
- styles.css  → Website design
- script.js   → API logic
- server.js   → Backend server
- .env        → API Keys

-------------------------------------------------------------
FEATURES
- Displays pets from PetFinder API
- Filters pets by age, gender, size, type
- Lets users adopt pets
- Stores adopted pets
- Beautiful UI + animations
- Fully responsive

-------------------------------------------------------------
TECHNOLOGIES USED
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- APIs (PetFinder, Dog API)
